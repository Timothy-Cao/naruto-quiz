# Brainstorm

Question-design scratchpad. **Not** part of the app — nothing here is built, shipped, or referenced by code. Treat these as raw ideas to argue with, prune, or promote later into:

- the live skill at `.claude/skills/naruto-quiz-authoring/SKILL.md` (canonized patterns), or
- actual quiz JSON in `data/quizzes/<slug>.json` (concrete questions).

Anything that survives review here gets a one-line summary added to the skill's Pattern Catalog and (optionally) a fully-authored example committed to a quiz. Until then it lives only here.

## Files

- `hard-logic-structures.md` — taxonomy of difficult-logic question shapes where the canon is load-bearing on the deduction. The bones, not the questions.
- `creative-question-ideas.md` — broader question ideas that aren't strict hard-logic but are unique, cool, or do something a normal quiz can't.
